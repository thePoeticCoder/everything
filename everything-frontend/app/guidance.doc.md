# Everything App — Information Document

This document contains detailed information about the **Everything** app. You can update or expand this document anytime as the app grows.

---

## 1. Overview

The **Everything App** is designed as an all‑in‑one powerhouse that provides solutions for multiple use cases — productivity, creativity, daily tasks, user engagement, and more.

The goal is to build a unified platform that simplifies life by offering various tools and features in a single place.

---

## 2. Main Hero Section (Landing Page)

The landing page serves as the first point of interaction for users. It contains:

### **Hero Background**

- Full‑width background image (`/images/hero-bg.webp`).
- Overlay for aesthetic visibility.
- Responsive design for small, medium, and large screens.

### **Hero Text Content**

- **Intro tagline:** "Let’s do the everything Here"
- **Main headline:** One solution, infinite possibilities.
- **Description:** Highlights the app as an adaptable, all‑in‑one productivity and creativity solution.

### **Rating Blocks**

- Displays ratings for:

  - Google (4.5)
  - LinkedIn (4.7)

- Uses the `RatingBlock` component.

### **Action Buttons**

- **Know the Developer** → `/about-me`
- **Get Free Trial** → `/create-account`
- Styled using Tailwind with hover effects.

## 3. Tech Stack (Current)

- **Framework:** Next.js (TypeScript)
- **Styling:** Tailwind CSS
- **Components:** Custom + shared blocks
- **Backend (Planned):** Prisma + SQLite (free) or free cloud database
